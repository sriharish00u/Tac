## Where to Download Free Images:

1. **Google Images** - Use "Free to use" filter
2. **Pixabay** - pixabay.com
3. **Unsplash** - unsplash.com
4. **Pexels** - pexels.com
5. **Freepik** - freepik.com
6. **Flaticon** - flaticon.com (for icons)
7. **Wikimedia Commons** - commons.wikimedia.org

## Tips for Using Images:

- Use high-resolution images (minimum 800x600px)
- Ensure images are royalty-free or properly attributed
- Compress images for faster loading
- Use consistent image sizes for grid layouts
- Add alt text for accessibility
